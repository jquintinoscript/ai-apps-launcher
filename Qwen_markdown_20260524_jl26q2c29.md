# 🚀 AI Apps Launcher

Gerenciador visual para suas aplicações de IA via linha de comando.

## ✨ Funcionalidades
- 🔍 Detecção automática de apps
- 🎨 Interface moderna e responsiva
- ⚡ Lançamento rápido com um clique
- 📂 Organização por pasta
- 🌐 Acesso via navegador

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/SEU-USUARIO/ai-apps-launcher.git
cd ai-apps-launcher

# Instale as dependências
pip install -r requirements.txt

# Execute
python app.py
